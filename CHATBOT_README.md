# Chatbot Δεξιοτήτων - Οδηγίες Χρήσης

## Περιγραφή
Αυτό το chatbot επιτρέπει την ανάλυση δεδομένων δεξιοτήτων για αποφοίτους με φιλτράρισμα βάσει τύπου σχολείου και φύλου, χρησιμοποιώντας το OpenAI API.

## Εγκατάσταση

### 1. Εγκατάσταση Dependencies
```bash
composer install
```

### 2. Διαμόρφωση OpenAI API Key

Ανοίξτε το αρχείο `app/config/config.php` και προσθέστε το OpenAI API key σας:

```php
'openai_api_key' => 'sk-your-api-key-here',
```

Για να λάβετε ένα API key:
1. Μεταβείτε στο https://platform.openai.com/
2. Συνδεθείτε ή δημιουργήστε λογαριασμό
3. Πηγαίνετε στο API Keys section
4. Δημιουργήστε ένα νέο Secret Key
5. Αντιγράψτε το key και τοποθετήστε το στο config.php

### 3. Εκκίνηση του Server

Μπορείτε να εκκινήσετε τον server με έναν από τους παρακάτω τρόπους:

**Χρήση PHP Built-in Server:**
```bash
composer start
```
ή
```bash
php -S localhost:8000 -t public
```

Το application θα είναι διαθέσιμο στο: http://localhost:8000

## Χρήση του Chatbot

### Βήμα 1: Επιλογή Φίλτρων
1. Μεταβείτε στο: http://localhost:8000/chat
2. Επιλέξτε τον τύπο σχολείου:
   - **ΓΕΝΙΚΟ (ΓΕΛ)**: Γενικό Λύκειο
   - **ΕΠΑΛ**: Επαγγελματικό Λύκειο
3. Επιλέξτε το φύλο:
   - **Άνδρας**
   - **Γυναίκα**
4. Επιλέξτε την περιφέρεια από τη λίστα (π.χ. Π.Ε. Αθήνας, Π.Ε. Θεσσαλονίκης κλπ.)
5. Πατήστε "Συνέχεια στο Chat"

### Βήμα 2: Συνομιλία με το Chatbot
1. Γράψτε την ερώτησή σας στο πεδίο εισόδου
2. Πατήστε "Αποστολή" ή πατήστε Enter
3. Το chatbot θα χρησιμοποιήσει τα φιλτραρισμένα δεδομένα για να απαντήσει

### Παραδείγματα Ερωτήσεων
- "Ποιες είναι οι κυριότερες ήπιες δεξιότητες που χρειάζονται οι απόφοιτοι;"
- "Ποια είναι η αναντιστοιχία στις τεχνικές δεξιότητες;"
- "Πόσο σημαντική είναι η επάρκεια στη διά βίου μάθηση;"
- "Ποιες δεξιότητες έχουν την υψηλότερη σημαντικότητα;"

## Χαρακτηριστικά

### Φιλτράρισμα Δεδομένων
Το chatbot φιλτράρει αυτόματα τα δεδομένα με βάση τις επιλογές σας:
- **ΓΕΝΙΚΟ**: Φιλτράρει δεδομένα που αφορούν αποφοίτους ΓΕΛ
- **ΕΠΑΛ**: Φιλτράρει δεδομένα που αφορούν αποφοίτους ΕΠΑΛ
- **Περιφέρεια**: Περιλαμβάνει στο context της συνομιλίας την επιλεγμένη περιφέρεια για πιο στοχευμένες απαντήσεις

Αυτό διασφαλίζει ότι στέλνονται στο OpenAI μόνο τα σχετικά δεδομένα, μειώνοντας το κόστος και βελτιώνοντας την ποιότητα των απαντήσεων.

### Τεχνολογίες
- **PHP Flight Framework**: Ελαφρύ micro-framework
- **OpenAI PHP Client**: Επικοινωνία με το OpenAI API
- **GPT-3.5-turbo**: Μοντέλο AI για απαντήσεις

## Αντιμετώπιση Προβλημάτων

### "OpenAI API key not configured"
- Βεβαιωθείτε ότι έχετε προσθέσει το API key στο `app/config/config.php`

### "Failed to get response"
- Ελέγξτε τη σύνδεσή σας στο Internet
- Βεβαιωθείτε ότι το API key είναι έγκυρο
- Ελέγξτε ότι έχετε επαρκή credits στο λογαριασμό OpenAI

### Το Composer δεν εγκαθιστά τα packages
```bash
composer clear-cache
composer install
```

## Κόστος API
Το chatbot χρησιμοποιεί το GPT-3.5-turbo που είναι οικονομικό:
- ~$0.002 ανά αίτημα (κατά προσέγγιση)
- Τα φιλτραρισμένα δεδομένα μειώνουν το κόστος
