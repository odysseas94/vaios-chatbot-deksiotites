# Chatbot Configuration Example

## Quick Setup Steps:

1. Open: app/config/config.php
2. Find the line: 'openai_api_key' => '',
3. Replace with your API key: 'openai_api_key' => 'sk-your-actual-api-key-here',

## Get Your OpenAI API Key:
Visit: https://platform.openai.com/api-keys

## Start the Server:
composer start
# or
php -S localhost:8000 -t public

## Access the Chatbot:
http://localhost:8000/chat
